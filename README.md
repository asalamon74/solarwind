# solarwind
Total solar eclipse corona/solarwind enhancer

```
Usage: solarwind.sh inputfile outputfile

Options:
  -h, --help                display this help
      --opendisk=radius     radius of the opendisk
      --closedisk=radius    radius of the closedisk

```
